Dieser Workspace dient als Startpunkt für die Entwicklung von eigenen Lösungen im Projektseminar Autonomes Fahren I und II.
Zum Starten muss dieses Git Repo geklont werden und die Informationen in den ``package.xml`` Dateien der einzelnen Pakete angepasst werden.

Eine *Erste-Schritte-Anleitung* `First_Steps.adoc` zu finden.

== Schnelleinstieg


Schnelleinstieg für bereits eingerichtete Computer. Es wird vorausgesetzt, dass die Rechner schon entsprechend der im https://git-ce.rwth-aachen.de/af/images[Image-Repository] gegebenen Ansible-Konfiguration eingerichtet sind.

(Die Anleitung `First_Steps.adoc` geht etwas genauer auf die einzelnen Schritte und das Fahrzeug ein.)

Klonen des Repos:

    git clone https://git-ce.rwth-aachen.de/af/ws-template

In den Ordner wechseln:

    cd ws-template

Projekt bauen:

    colcon build --symlink-install

Sourcen des Workspaces:

    source install/setup.bash

Zum Starten aller Pakete:

    ros2 launch psaf_launch main_psaf1.launch.py

Zum Starten eines Pakets:

    ros2 launch <package_name> <launch_file>             # Allgemein
    ros2 launch psaf_firststeps firststeps.launch.py     # Beispiel

[IMPORTANT]
Die Pakete nicht mit dem Befehl `ros2 run <package_name> <node_name>` starten, da diese ansonsten nicht korrekt funktionieren.
